# 🐍 Snake Water Gun Game

A simple Snake Water Gun game built using Python.

## About Project

This is a command-line game where:
- Snake drinks Water 🐍
- Water destroys Gun 💧
- Gun kills Snake 🔫

The computer randomly selects one option and the user plays against the computer.

## Technologies Used

- Python
- Random Module
## How to Run

1. Clone this repository
git clone your-repository-link

2. Run the file
python game.py

## Game Rules

| Choice | Value |
|---|---|
| Snake | 1 |
| Water | -1 |
| Gun | 0 |

## Example Output

Enter your choice: s

You chose snake
Computer chose water

You win!
